# DATASET_LICENSE_AUDIT.md - Commercial-Safety Audit

Audit date: 2026-05-18

This is an engineering guardrail, not legal advice. For a paid or public release, keep a human legal review in the release checklist.

## Policy

Default mining commands should only include sources that are reasonably safe for commercial experiments. Sources with non-commercial terms, share-alike obligations, unclear licensing, or closed-model distillation terms are removed from built-in project pipelines.

## Release Checklist

Before releasing a commercial SigerLM model:

1. Save each dataset card snapshot or commit hash used for training.
2. Keep `data/*report.json` mining reports with the model artifact.
3. Add attribution for MIT, Apache 2.0, CC BY, and source-specific terms.
4. Do not re-add removed sources to commercial builds unless legal review approves them.
